# Análisis de Pórticos 2D - Streamlit App

Esta aplicación permite generar estructuras de pórticos 2D, definir sus condiciones de frontera y cargas, y visualizar los resultados mediante análisis estructural.

## 📁 Estructura de carpetas

- `Analizadores/`: Código para resolver el análisis estructural.
- `Calculadoras/`: Cálculo de momentos y otros resultados internos.
- `Generadores/`: Scripts para generar estructuras y archivos JSON.
- `Ejemplos/`: Archivos de ejemplo ya preparados (`P1.json`, etc).
- `app.py`: Punto de entrada principal de la aplicación Streamlit.
- `interfaz.py`: Código para el diseño de la interfaz gráfica.
- `estructura.json`: Archivo que se genera con los datos ingresados por el usuario.
- `requirements.txt`: Lista de librerías necesarias para correr la app.
